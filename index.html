Yeoman is a robust and opinionated set of tools, libraries, and a workflow that can help developers quickly build beautiful, compelling web apps.

http://yeoman.io

Spend less time on process and more time focusing on building beautiful web applications.

How?

→ Lightning-fast scaffolding
→ Automatically lint your scripts
→ AppCache manifest generation
→ PhantomJS Unit Testing
→ Awesome Image Optimization
→ Built-in preview server

good! 

$ yeoman init

Initialize and scaffold a new project using generator templates

You get

  • HTML boilerplate
  • Twitter Bootstrap
  • Require.js
  • you can start building on your project 

$ yeoman server

Starts a localserver

Has live reload

Keeps tracks of *.css and *.js

$ yeoman build

Concatenates scripts

Optimises images

Runs tests

Creates manifest.appcache

$ yeoman server:dist
